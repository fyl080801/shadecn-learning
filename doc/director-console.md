# 3D导演台 — 需求文档

## 1. 概述

3D导演台是一个基于 Three.js 的场景编排工具，用于在 3D 场景中摆放、调整图形（角色）和机位，并支持在"导演视角"与"机位视角"之间切换预览，服务于内容拍摄前的场景调度与运镜预演。

本文档基于设计原型截图整理，并结合当前迭代的功能范围做出以下调整（不做的部分本期不实现）：

- 场景面板：仅列出场景中的图形，**不做搜索、不做分组**
- 悬浮工具栏：**只保留**变换模式切换（移动/旋转/缩放）、添加角色、添加机位三类操作，其余（截图、导出、全屏等）本期不做
- 右侧属性面板：根据"是否选中图形"切换展示内容（场景属性 / 图形属性）

## 2. 整体布局

```
┌───────────────────────────────────────────────────────────┐
│  3D导演台            [导演视角 | 机位视角]                    │
├───────────┬───────────────────────────────────┬─────────────┤
│           │                                   │             │
│  场景面板  │        3D 场景渲染区域              │  属性面板    │
│ (图形列表) │                                   │ (场景/图形)  │
│           │        ┌─────────────────┐         │             │
│           │        │  悬浮工具栏       │         │             │
│           │        └─────────────────┘         │             │
└───────────┴───────────────────────────────────┴─────────────┘
```

- 左侧：场景面板（固定宽度侧边栏）
- 顶部：视角切换 Tab
- 中间：3D 场景渲染区域，内部悬浮工具栏
- 右侧：属性面板（场景属性 / 图形属性二选一，互斥展示）

## 3. 功能模块

### 3.1 顶部视角切换

- 提供两个 Tab：**导演视角**、**机位视角**
- 导演视角：自由相机，用于整体编排场景、拖拽/摆放图形和机位
- 机位视角：切换为某个已添加机位的相机视角，用于预览该机位实际取景效果
- 切换 Tab 只影响渲染区域使用的相机与交互方式，不影响左右两侧面板的内容

### 3.2 左侧场景面板

- 展示当前场景中的图形列表（角色/机位等可选中对象）
- **不提供搜索框，不做分组/分类**，按添加顺序平铺展示
- 每一项展示：
  - 图形名称（如"角色A"）
  - 可见性切换（显示/隐藏，眼睛图标）
  - 锁定切换（锁定后禁止在视口中被选中/拖动，锁图标）
- 点击列表项 = 在 3D 视口中选中对应图形，联动右侧属性面板切换为"图形属性面板"
- 再次点击已选中项 / 点击空白处 = 取消选中，右侧面板回退为"场景属性面板"

### 3.3 中间 3D 场景渲染区域

- 渲染当前场景（模型、全景背景等）
- 支持视口内直接点选图形进行选中（与左侧列表选中状态双向同步）
- 选中图形后，视口内显示对应的变换控制器（Gizmo），控制器形态随当前变换模式（移动/旋转/缩放）变化
- 保留视角小工具（轴向指示器 + 重置视角），用于快速复位导演视角相机

#### 3.3.1 悬浮工具栏（简化版）

本期悬浮工具栏**只包含以下三类控制**：

1. **控制按钮（下拉选择）**：切换当前变换操作模式，下拉选项为：
   - 移动（Translate）
   - 旋转（Rotate）
   - 缩放（Scale）
   - 仅在选中图形时生效；未选中图形时该控件禁用或隐藏
2. **添加角色（图形）**：向场景中新增一个图形对象，新增后自动加入左侧场景面板列表，并默认选中
3. **添加机位**：向场景中新增一个机位（相机）对象，新增后自动加入左侧场景面板列表；机位作为一种特殊图形参与列表展示与选中逻辑，可在"机位视角"中被选择切换预览

不在本期范围内的原型功能（暂不实现）：多机位快速切换按钮组、截图、拍照/快照、导出、全屏。

### 3.4 右侧属性面板（二选一，互斥）

#### 3.4.1 未选中任何图形 → 3D场景属性面板

- **场景缩放**：滑块 + 百分比数值
- **场景平移**：X / Y / Z 数值输入
- **场景旋转**：X / Y / Z 数值输入
- **全景背景**：
  - 已连接的全景图（缩略图展示，可更换）
  - 天空颜色（拾色器 + hex 输入）
- **全景球**：
  - 水平旋转（滑块 + 角度数值）
  - 球形半径（滑块 + 数值）
- **角色标签开关**：控制视口中是否显示图形名称浮动标签

#### 3.4.2 选中某个图形 → 图形（角色）属性面板

- 图形名称（展示/可编辑）
- 变换属性：
  - 位置 X / Y / Z
  - 旋转 X / Y / Z
  - 缩放 X / Y / Z（或统一缩放）
- 可见性 / 锁定状态（与左侧场景面板中的状态联动）
- 若为机位类型图形，额外展示机位专属属性（如视场角 FOV、朝向目标等，具体字段视机位数据模型确定）

> 属性面板与视口内 Gizmo、左侧列表三者的选中/数值状态需保持实时双向同步：任一处修改，其余两处同步更新。

## 4. 交互状态与联动关系

| 操作来源 | 触发结果 |
|---|---|
| 点击左侧列表项 | 视口中对应图形高亮选中 + 右侧切换为图形属性面板 |
| 视口中点击图形 | 左侧列表对应项高亮 + 右侧切换为图形属性面板 |
| 视口中点击空白 / 再次点击已选图形 | 取消选中 + 右侧回退为场景属性面板 |
| 工具栏"添加角色/添加机位" | 新增对象、加入左侧列表、自动选中新对象、右侧切换为图形属性面板 |
| 切换变换模式下拉 | 视口 Gizmo 形态切换（移动/旋转/缩放），仅选中图形时可操作 |
| 顶部切换"导演视角/机位视角" | 渲染区域相机与交互模式切换，不影响左右面板当前展示内容 |
| 左侧列表切换可见性/锁定 | 视口中对应图形显示/隐藏，或禁止被选中和拖动 |

---

## 5. 本期不做（Out of Scope）

- 场景面板搜索框
- 场景面板分组/分类
- 悬浮工具栏中的截图、拍照、导出、全屏等按钮
- 多机位并排预览/快速切换按钮组
